# High Priority (MVP):

* As a Mentor, I want to add a new kapatid with name, classification, mentor, initial progress stage, and details so that I can start tracking them immediately.
* As a Mentor, I want to view a list of all my assigned kapatids with their current progress stage so that I can quickly see who needs follow-up.
* As a Mentor, I want to update the progress stage of a kapatid (e.g. Pre-FIC → FIC1) so that the record stays accurate.
* As a Mentor, I want to add/edit details/notes for a kapatid so that I can record important life updates or prayer points.
* As a Mentor, I want to search or filter kapatids by name, classification, or progress stage so that I can find them easily.
* As a Mentor, I want to see a simple dashboard showing total count per progress stage so that I can monitor the overall spiritual growth of my group.

# Medium Priority (Post-MVP):

* As a Mentor, I want to assign or change the mentor of a kapatid.
* As an Admin, I want to manage all mentors and users.
* As a Kapatid, I want to view my own progress and details (read-only).
* As a Mentor, I want to export the list (CSV/PDF) for reporting.

# Acceptance Criteria example (para sa unang story):

* Fields: Full Name (required), Classification (dropdown), Mentor (dropdown or text), Progress Stage (dropdown with the 5 stages), Details (textarea).
* After adding, dapat lumabas agad sa list.
* May validation para hindi maulit ang pangalan o walang laman na required fields.
